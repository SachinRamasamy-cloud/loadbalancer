# Terminology

| Term | Meaning |
|---|---|
| Backend | Service receiving forwarded requests |
| Load balancer | Service selecting a backend |
| Health check | Probe determining backend availability |
| Failover | Selection of an alternative backend |
| Retry | Reattempt under a defined policy |
| Weight | Relative routing share |
| Upstream | Backend behind the load balancer |
| Orphan container | Container absent from the current Compose file |
