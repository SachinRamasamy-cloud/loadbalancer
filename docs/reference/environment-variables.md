# Environment Variables

| Variable | Purpose |
|---|---|
| `ALLOWED_BACKEND_HOSTS` | Restricts valid backend hostnames |
| `SEED_BACKENDS_JSON` | Initial backend registry |
| `BACKEND_NAME` | Simulated backend name |
| `RESPONSE_DELAY_MS` | Simulated latency |
| `FAILURE_RATE` | Simulated request failure probability |
| `HEALTH_FAILURE` | Optional simulated health failure |
